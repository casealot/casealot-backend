## ⚙ 개발 환경

- 운영체제 :  Windows
- 통합개발환경(IDE) : IntelliJ
- JDK 버전 : JDK 17
- 데이터 베이스 : Mysql
- 빌드 툴 : Gradle-7.6.1
- CI/CD : jenkins
- 관리 툴 : GitHub
- 웹 호스팅 : AWS(EC2, S3, RDS, ROUTE 53)

    - React
